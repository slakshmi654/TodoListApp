# Python To-Do List Application

## Description
A beginner-friendly Python console-based To-Do List application built using PyCharm.

## Features
- Add tasks
- View tasks
- Remove tasks
- Save tasks to file

## Technologies Used
- Python
- PyCharm
- Git & GitHub

## How to Run
1. Clone the repository
2. Open in PyCharm
3. Run todo.py
